NP ScrollUp is an awesome jQuery scroll to top plugin.

== Description ==

NP ScrollUp is an awesome jQuery scroll to top plugin for your WordPress website. This plugin will enable awesome scroll to top button with jQuery effect.

It has a lot of options in option panel of this plugin. You can change icon color, icon hover color, background color, background hover color, padding of the icon, icon size, icon frame, icon style, icon position, appearance animation, scroll to top text and icon border from options panel of this plugin. Below you can see the list of the options name of the options panel of this plugin.

* Icon color
* Icon Hover color
* Background color
* Background Hover color
* Padding
* Icon Size
* Icon Frame
* Icon Style
* Icon Position
* Appearance Animation
* Scroll To Top Text
* Scroll Up Icon Border

To see demo & documentation [click here.](http://np-foundation.twomini.com/plugin/np-scrollup "Demo & Documentation")
